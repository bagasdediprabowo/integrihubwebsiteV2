IntegriHub API
POST /v1/messages/template

Kirim Pesan Template (WABA)

Endpoint ini digunakan untuk mengirim pesan template *WhatsApp Business API* yang telah disetujui oleh Meta (Approved Templates). Sangat cocok untuk notifikasi OTP, pengingat tagihan (Invoice), dan *alert* sistem.

Parameter Headers

Authorization Required

Gunakan Bearer Token (API Key) dari Dashboard SaaS IntegriHub Anda.

Contoh Request (cURL)

curl -X POST https://api.integrihub.my.id/v1/messages/template \
-H "Authorization: Bearer sk_live_your_api_key" \
-H "Content-Type: application/json" \
-d '{
  "to": "+6285711164332",
  "template_name": "invoice_notification",
  "language": "id",
  "components": [
    {
      "type": "body",
      "parameters": [
        {"type": "text", "text": "Bagas Dedi Prabowo"},
        {"type": "text", "text": "INV-2025-001"}
      ]
    }
  ]
}'

Contoh Response (JSON)

{
  "status": "success",
  "message_id": "wamid.HBgNNjI4NTcxMTE2NDMzM...",
  "delivered_to": "+6285711164332"
}
